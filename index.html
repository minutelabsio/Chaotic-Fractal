Choose a number just for fun,
between (but not including) 0 and 1.
Subtract it from 1 to get another,
and multiply it by the other.
0.50 x ( 1 - 0.50 )
0
1
Next we need to choose one more,
this time between 3 and 4.
Call it "r" so we don’t lose track.
(For future iterations we’ll need it back)
3.30 x 0.50 x ( 1 - 0.50 )
3
4
The final expression is not so complex:
"r" times "x" times "1 - x"
The result becomes "x" reborn!
Repeating the process will adorn...
3.30 x 0.50 x ( 1 - 0.50 ) = 0.000
For every "r" begin with an "x",
plot that sequence, and move on to the next...
The graph it makes is a pleasing sight.
Self-similar patterns to stare at all night.
